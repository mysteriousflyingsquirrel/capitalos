# Capitalos

A unified wealth management platform designed to track total net worth, inflows, outflows, and asset allocations across multiple financial universes (Crypto, Stocks, Real Estate, Cash).

## Features

- **Dashboard**: Total net worth, asset breakdown, inflow/outflow charts
- **Cashflow**: Categorized lists, filters, monthly flow visualization
- **Net Worth**: Long-term value graph, composition pie, and trend insights
- **Investing**: Holdings by platform and asset, profit/loss summary
- **Settings**: Account management, data sync, tax reports

## Tech Stack

- **Frontend**: React + Vite + TailwindCSS (PWA)
- **Backend**: Google Apps Script (for API bridge and Drive access)
- **Data Storage**: Google Drive JSON files (private to user)
- **Auth**: Google OAuth 2.0

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Design System

The application uses a dark, galactic theme with:
- Primary Background: #050A1A (Dark Galaxy Blue)
- Accent Colors: #B87333 (Bronze Gold) and #DAA520 (Goldenrod)
- Typography: Inter, sans-serif

See `Capitalos_UI_Style_Guide.txt` for complete design specifications.
